# fastly_request_setting

Creates and updates a Fastly request setting.

## Actions

* `:create`

## Properties

* `api_key`, `username`, `password` - Fastly authentication.
* `service` - Fastly service name.
* `request_condition`, `force_ssl`, `force_miss`, `bypass_busy_wait`, `default_host`, `hash_keys`, `max_stale_age`, `request_action`, `x_forwarded_for` - Request settings.
